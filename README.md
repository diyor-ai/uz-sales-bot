## ⚡ The Problem
Uzbekistan's Telegram marketplace is huge, but **human delay kills sales**. 
- ❌ **Average Response Time:** 45 - 180 minutes.
- ❌ **Customer Drop-off:** 60% of users leave if not answered in 5 minutes.
- ❌ **Manual Errors:** Typo in prices and inventory leads to lost trust.

## 💎 The Solution: Uz-Sales-Bot
An automated sales agent that acts as a bridge between a **Google Sheet Inventory** and a **Telegram Interface**. 

### 🌟 Core Features
- **Instant Search:** Find any product in milliseconds using keyword matching (Day 2).
- **Headless CMS:** Shop owners manage products directly from **Google Sheets**.
- **Human-Like Order Flow:** Streamlined 3-step checkout process.
- **Admin Alerts:** Instant Telegram notifications for every new order.
